# Hospital Management App

This is some hospital website that can be used as management system, in which patients can book appointment with the hospital. Then, every doctors have their own accounts. This doctors as **Admins**, can manage incoming appointment registered in the system.

## Tech Stacks

- Laravel
- Livewire
- MySQL
- Tailwind + Flowbite plugin

## What I learnt

This project sorely for learning Laravel + Livewire framework purpose. For that, this project has given me at great deal of understanding on how to manage server-side reactivity using PHP.


### Install Composer Dependencies

```bash
composer install
```

### Migrate Database and Seeding

```bash
php artisan migrate:fresh --seed
```

### Install Node Dependencies

```bash
npm install
```

### Run Server

```bash
# it will live on localhost:8080
php artisan serve
```

### Run Node Server

```bash
npm run dev
```

### Open the website

Open your browser, enter `http://localhost:8080`
